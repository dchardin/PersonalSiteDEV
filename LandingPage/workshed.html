TEXT-TO-PIC


There are times when I need to send one-time passcodes to friends and colleages via SMS/Text messaging. Since the recipients are often receiving them on their smartphones, the message can be difficult to read. It makes more sense to send the message as a large image with a dark background, making it easier to read. Kabir Shah's work on his "Alter" webtool provided the basis for this project. It is a simple web page that will take text input, pass it to javascript, which will then make use of the HTML5 canvas API to return an image. You can try it out


CamelCityCouriers.com


This is a single page app that I developed for a fictitious company I made up as an Uber competitor. It features a parralax scrolling effect accomplished with CSS selectors from a grid-based boiler plate, as well as a few simple javascript features. You can view the page and view the source on Github
